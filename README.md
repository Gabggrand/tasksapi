# Tasks API – Spring Boot REST API

Uma API simples e rápida criada com **Spring Boot**, permitindo criar, listar, buscar, atualizar e deletar tarefas (Tasks).  
Ideal para estudos, portfólio ou integrações simples.

---

## Tecnologias utilizadas

- **Java 17+**
- **Spring Boot**
- **Spring Web**
- **Postman** para testes
- **API REST**
